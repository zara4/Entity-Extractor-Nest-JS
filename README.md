  <p align="center">A package which you can use to extract entities from a text file to a given path of a text file</p>
    <p align="center">
  
## Installation

```bash
$ npm install cli-console-entity-extractor
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm start

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
